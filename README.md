
# Make your own Language - Basic C-Like Compiler
 An attempt to write simple Compiler, Assembler building blocks & a Virtual Machine to run the compiled code.

Steps: 
- LexicalAnalyzer
- BNF_To_C++
- SyntaxAnalyzer
- CodeGenerator
- VMInterpreter
